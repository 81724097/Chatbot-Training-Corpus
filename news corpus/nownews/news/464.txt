在二月中的2017 OSAKA AUTOMESSE大阪改裝車展中，邢男再次見到了日前剛在一月東京改裝車展中，算是「亞洲」首次曝光的十代Civic Type R Prototype(原型車)，但就在筆者撰寫本文的同時，終於看到本田原廠正式發布十代Civic Type R的「量產版」車型，即將在下個月的日內瓦車展中正式發表重要新聞！在此還是先來看看這部超帥氣的原型車。	（圖／車訊網提供）相信你會跟我一樣，當去年首次見到本車曝光時，它那原廠讓人「帥到分手」的超霸氣外觀，包含像是鈦灰色塗裝、燻黑頭燈、引擎蓋進氣孔、全新水箱護罩、巨幅開口前/後氣壩、碳纖維材質下巴/側裙/後下分流器/寬體外拓輪拱葉子板、紅色LOGO/下巴/側裙/框邊、二大一小排氣尾管與245mm胎寬+20吋圈等配置特徵，車迷們一定祈求上天保佑，三月正式量產版車型亮相時，一定要長得就是這幅模樣！（圖／車訊網提供）	（圖／車訊網提供）而車迷們最關注的十代Civic Type R量產版車型搭載動力規格，先前國內、外媒體報導不一，有人推測應會沿用上一代FK2首次搭載本田全新研發的2.0升i-VTEC Turbo渦輪動力(K20C1引擎：310hp/40.8kgm)，有人則推測會改用十代一般款Civic所搭載1.5T引擎(美規174hp/22.4kgm)的「高增壓版」特調動力，直到這兩天本田原廠正式確認，十代Civic Type R將沿用FK2的2.0T動力+6MT六速手排變速箱，至於出力數據仍未公布，但以前代FK2坊間改裝廠透過電腦程式Remapping即可獲得30~50hp不等的提升幅度來看，新Type R的原廠動力保守估計至少可達345hp！一來便可追上紐柏林最速FF挑戰者Ford Focus RS(350hp)，也可藉此擺脫難纏的對手Golf GTI Clubsport S(310hp)啊！	（圖／車訊網提供）本田原廠表示，今年夏天開始量產的十代Civic Type R(包含Civic一般款五門Hatchback車型)將在英國Swindon工廠上線，Civic Sedan四門車型則將在日本國內琦玉工廠生產。特別值得一提的是，這次全新的十代Civic Type R也將首次在北美販售，儘管過去Civic在當地頗受年輕車迷與改裝玩家喜愛，但始終只有一般款Sedan、Hatchback與Si車型，所以目前美國當地不論媒體、改裝廠與本田迷們，都對Type R的上市滿心期待！至於日本國內十代Type R仍將維持英國生產「逆輸入」回國銷售的模式，但從這代開始將會取消前代700輛的「限量預購」銷售策略。而國內車迷最關心的台灣總代理動作呢？直到目前官方說法十代Civic引進尚在密切評估中，當然這是指搭載2.0 NA(158hp)或1.5T(174hp)渦輪動力的一般車型，至於引進版本會是美規或歐規？四門或五門？以目前十代Civic全球車型外觀設計差異並不算大的情況下，總代理引進美規或歐規，最終以美元與歐元匯率的成本考量將會是決策因素之一。至於四門或五門，老實說，十代Civic五門Hatchback的外型設計頗為討喜，若能將進口價格定在新台幣100萬元上下，應可獲得國內消費者青睞。至於Type R車型，建議你還是找間曾經引進販售FK2九代Civic Type R的貿易商比較快，推估價格也將超過新台幣210萬元囉！	（圖／車訊網提供）
